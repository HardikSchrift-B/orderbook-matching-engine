# Low-Latency Limit Order Book Matching Engine

A simplified exchange-style matching engine built in C++.

## Features

- Price-time priority matching
- Buy / Sell limit orders
- Partial fills
- Order cancellation
- Real-time order book display
- Throughput benchmark: 1M+ simulated orders/sec

## Technologies

- C++
- STL
- map
- queue

## Use Case

Simulates how electronic exchanges process market orders.
